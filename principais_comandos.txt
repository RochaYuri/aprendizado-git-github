git add . (adiciona todos os arquivos na fila de commit)
git commit -m '' (realiza o commit com a mensagem de descrição)
git merge <file> (Realiza o merge do arquivo passado na branch atual)
git checkout <branch> (altera para a branch passada)
git checkout -b <branch> (cria uma branch, e altera para ela)
git clone <link> (Clona o repositório passado)
git status (Verifica o status da fila do commit)
git log (Verifica o log de todos os commits)
git push origin <branch> (Realiza o envio do commit para a branch passada na origem (normalmente github))
git add remote origin <link> (Adiciona e configura a nova origem)
